# Changelog

## [Unreleased]

### Added
- Added `:BufferPluginInfo` command and `<leader>uP` mapping to inspect plugins active in the current buffer
- Added `<leader>er` mapping to reveal current file in NvimTree
- Added `<leader>cr` (normal mode) and `<leader>cr` (visual mode) mappings for Conform to format current line or selection
- Added `<leader>gt` and `<leader>gT` mappings to show unified git diff popup for current file
- Added `<leader>md` mapping to preview Mermaid diagrams from markdown inside Neovim
- Added support for Clojure formatting with `zprint` (replaces `cljfmt`)
- Added automatic clipboard provider detection for Wayland (`wl-clipboard`) and X11 (`xclip`)
- Added right padding only for buffers with less than 5000 lines to improve performance

### Changed
- Changed default formatter for Clojure from `cljfmt` to `zprint`
- Changed markdown preview feature to Mermaid diagram preview using `diagram.nvim` and `image.nvim`
- Changed auto-close logic to only trigger on destructive events (BufDelete, BufWipeout, WinClosed, TabClosed) to reduce latency
- Changed NvimTree to not update root directory on file focus changes to avoid unnecessary directory changes
- Changed NvimTree to not resize window when opening files to maintain stable layout
- Changed Aerial to not auto-close when backend is missing and disabled automatic opening on first symbols
- Changed Aerial to prefer explicit `:AerialOpen right` command over API calls
- Changed Sidekick to disable NES (Nintendo Entertainment System) integration by default
- Changed Sidekick to use simpler window management without complex tab state tracking
- Changed Sidekick to always open in right split layout when toggled
- Changed Conjure to make auto-evaluation on InsertLeave optional (disabled by default)
- Changed Conjure to use async port checking to avoid UI freezes
- Changed Copilot to check Node.js version asynchronously during startup
- Changed git diff popup to use async commands to prevent UI blocking
- Changed notify ticker to use shorter timeouts and simpler marquee logic
- Changed history view to limit number of displayed entries and trim long lines
- Changed Lualine to disable progress bar integration to reduce statusline churn
- Changed Lualine to guard against plugins changing global statusline settings
- Changed Yanky to disable automatic clipboard sync to prevent UI blocking
- Changed auto-close to exclude buffers with special filetypes (NvimTree, aerial, alpha, help, qf)
- Changed right padding to trigger on BufWinEnter and FileType events instead of BufEnter/WinEnter
- Changed indent guides to skip odd/partial indentation lines in Clojure files
- Changed Vim Sleuth to keep Clojure indentation at 2 spaces by default

### Fixed
- Fixed alpha dashboard crashes by guarding against stale window IDs
- Fixed alpha dashboard footer to update after Lazy startup events
- Fixed alpha dashboard to properly handle small terminal sizes
- Fixed Aerial auto-open logic to be more deterministic and avoid duplicate panels
- Fixed Sidekick to properly close Aerial panel when opening
- Fixed Conform to properly handle format-on-save for various buffer types
- Fixed git diff popup to work when current window is invalid
- Fixed notify ticker to properly handle fast event contexts
- Fixed clipboard integration to work properly on Wayland systems
- Fixed NvimTree to properly focus main window after opening
- Fixed auto-close to properly handle terminal buffers and special windows
- Fixed right padding to not apply to very large buffers

### Removed
- Removed markdown-preview.nvim plugin
- Removed drop.nvim plugin
- Removed nvim-scrollbar plugin
- Removed duplicate NvimTree autocommands from global config
- Removed progress bar integration from Lualine
- Removed complex Sidekick tab state management
- Removed automatic Aerial opening on first symbols

### Deprecated
- Deprecated `<leader>mp` mapping (markdown preview) in favor of `<leader>md` (Mermaid preview)
- Deprecated `<leader>cf` mapping (generate function docs) in favor of `<leader>cF` to avoid conflict with format mapping
