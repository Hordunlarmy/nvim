# Changelog

## [Unreleased]

### Added
- Added `.clj-kondo/` to `.gitignore`
- Added new plugin `alpha-ascii.nvim` for random ASCII art headers in dashboard
- Added new keymap `<localleader>rr` to reload all modified namespaces in Conjure (tools.namespace refresh)
- Added new keymap `<F2>` to clear Conjure log in both normal and insert modes
- Added auto-evaluation of Clojure buffers on InsertLeave in Conjure
- Added new user commands: `AlphaFindFiles`, `AlphaRecentFiles`, `AlphaLiveGrep` for dashboard actions

### Changed
- Improved auto-close logic to handle more filetypes and edge cases
- Updated multiple plugin versions (LuaSnip, blink.cmp, nvim-treesitter)
- Refactored dashboard UI with cleaner layout, random ASCII art header, and responsive design
- Improved dashboard key mappings and telescope integration with centered popups
- Enhanced Conjure integration with better log management and automatic evaluation
- Improved Treesitter configuration with fallback handling and disabled Clojure highlighting
- Simplified which-key telescope integration by using shared `telescope_float` module

### Fixed
- Fixed auto-close not triggering when switching between certain buffer types
- Fixed dashboard not properly handling small terminal windows
- Fixed Conjure log not properly clearing in some cases
- Fixed Treesitter configuration to handle potential module loading failures
- Fixed dashboard buttons not being properly aligned in different window sizes
