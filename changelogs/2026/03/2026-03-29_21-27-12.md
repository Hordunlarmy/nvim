# Changelog

## [Unreleased]

### Removed
- Removed Avante AI integration (plugin and all related features)
- Removed `<leader>ar` keybinding for reloading Avante
- Removed Avante-specific documentation from README

### Changed
- Updated multiple plugin versions (LuaSnip, aerial.nvim, alpha-nvim, blink.cmp, etc.)
- Improved git diff popup functionality with better error handling
- Enhanced API key management with blocked environment variables protection
- Improved notification system with better routing and history
- Updated lualine status bar with better progress indicators and navigation
- Improved telescope keymap picker with better centering
- Enhanced conform.nvim formatting with auto-format on escape in insert mode
- Improved LSP hover documentation centering

### Added
- Added new `<leader>gb` keybinding for full-buffer git blame
- Added new `<leader>dt` keybinding for unified git diff popup
- Added new `<leader>db` keybinding for toggling diagnostics
- Added new environment variable blocking system to prevent GitHub tokens in Neovim
- Added new notification history popup with `<leader>uh` and `<leader>uH`
- Added new sidekick AI integration with multiple keybindings
- Added new winbar navigation with nvim-navic integration
- Added new progress marquee animation for better visual feedback
- Added new drop effect toggle with `<leader>uD`

### Fixed
- Fixed notification routing to ensure messages appear in statusline
- Fixed plugin reload functionality to maintain environment variable blocking
- Fixed git diff popup to work reliably with gitsigns
- Fixed LSP hover window positioning
- Fixed telescope keymap picker centering in current window
- Fixed conform.nvim formatting to work in more contexts
