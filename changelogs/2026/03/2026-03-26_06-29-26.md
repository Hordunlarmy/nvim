# Changelog

## [Unreleased]

### Added
- Added `vim.loader.enable()` call in `init.lua` for faster startup when available
- Added new plugins:
  - `cmp-conjure` for Clojure completion
  - `conform.nvim` as a replacement for `formatter.nvim`
  - `conjure` for Clojure REPL integration
  - `nvim-paredit` for Clojure structural editing
- Added Clojure language support with:
  - `clojure_lsp` configuration
  - Clojure-specific diagnostics toggles (`<leader>dk`)
  - Clojure cache reset command (`<leader>dK`)
  - Custom Aerial symbol handling for Clojure macros
- Added comprehensive diagnostics control system:
  - Buffer-local toggle (`<leader>dt`)
  - Global toggle with persistence (`<leader>dT`)
  - Clojure-specific toggle (`<leader>dk`)
  - Clojure cache reset command (`<leader>dK`)
- Added new keymaps:
  - `<leader>fC` to search Clojure-specific keymaps
  - `<A-f>` and `<M-f>` for current buffer search
  - `<leader>Q`, `<leader>W`, `<leader>WQ` as instant alternatives to `qq`, `ww`, `wq`
  - `<localleader>` mappings for Conjure (Clojure REPL)

### Changed
- Improved diagnostics display with minimal icons (E, W, H, I)
- Updated plugin versions:
  - `lazy.nvim` updated
  - `nvim-treesitter` branch changed from master to main
- Refactored diagnostics system to be more configurable and persistent
- Improved Aerial configuration with:
  - Better Clojure support
  - More reliable auto-open logic
  - Better error handling
- Improved notification system with better dismiss behavior
- Improved border styling with consistent white borders across all floating windows
- Improved keymap organization in which-key
- Improved filetype-specific indentation handling
- Improved auto-close logic to be more reliable
- Improved NvimTree integration with diagnostics toggle awareness
- Improved Copilot integration with better Tab handling
- Improved LSP configuration with better error handling

### Fixed
- Fixed issue with bufferline being removed (replaced with cokeline which is now disabled)
- Fixed various deprecation warnings
- Fixed diagnostics display issues
- Fixed notification display and dismissal
- Fixed floating window borders and padding
- Fixed auto-close logic to properly handle various buffer types
- Fixed mode colors to properly update on colorscheme changes
- Fixed right padding to not interfere with text formatting
- Fixed various plugin configuration issues

### Removed
- Removed `bufferline.nvim` plugin (replaced with disabled cokeline)
- Removed `formatter.nvim` (replaced with conform.nvim)
- Removed various deprecated configurations and workarounds

### Security
- Added proper error handling for various operations
- Improved plugin loading safety checks
