## Lualine
Lualine is the statusline. It shows mode, branch, diagnostics, file name, and Copilot status.

## Bufferline
Bufferline shows open buffers in a tab-like bar. It integrates with NvimTree and Aerial via offsets. It is configured to keep backgrounds transparent and always show buffer icons.

## Notify and Noice
nvim-notify provides floating notifications. noice.nvim improves command line UI and message handling. The config is tuned to avoid stuck notifications and to reduce popup noise.

## Dressing
Dressing improves UI for input and select prompts.

## Transparent
Transparent makes the main editor background transparent.

## Twilight
Twilight dims inactive code for focus.

## Alpha dashboard
Alpha provides a start screen (dashboard).
