## Mason
Manages LSP servers and external tools (formatters, linters). It is used by the LSP and formatting layers.

## Devicons and Plenary
Devicons provides file icons for UI plugins, and Plenary is a shared utility library used by Telescope and others.

## BQF, Scrollbar, Todo Comments
Small UI enhancements for quickfix list, scrollbars, and todo highlighting.

## Markdown preview
Allows previewing markdown in a browser.

## Theme
Provides the main colorscheme.

## Project.nvim
Disabled in this setup to avoid conflicts with NvimTree.

## Refactoring and Neogen
Provides code refactor helpers and docstring generation.
